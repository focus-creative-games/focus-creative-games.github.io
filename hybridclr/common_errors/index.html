# йҒҮеҲ° MissingMethodException xxx й”ҷиҜҜ

еҰӮжһңжңӘеҮәзҺ°AOT generic methodзҡ„еӯ—зңјпјҢеҲҷиҝҷжҳҜunityд»Јз ҒиЈҒеүӘеј•иө·зҡ„еҮҪж•°дёўеӨұпјҢдҪ дҪҝз”Ёеёёи§„зҡ„йҒҝе…Қunityд»Јз ҒиЈҒеүӘзҡ„ж–№ејҸеӨ„зҗҶеҚіеҸҜгҖӮ

еёёи§„дёӨз§ҚеҒҡжі•

  • еңЁlink.xml preserveиҝҷдёӘеҮҪж•°гҖӮ
  • дё»е·ҘзЁӢдёӯжҳҫејҸеёҰдёҠзұ»жҲ–иҖ…еҮҪж•°зҡ„дҪҝз”ЁпјҢеҰӮhybridclr_trial (opens new window)дёӯAssets/Main/hybridclrLib/RefTypes.csжүҖеҒҡзҡ„йӮЈж ·гҖӮ

# йҒҮеҲ° Unity: TypeLoadException xxx й”ҷиҜҜ

еҗҢж ·жҳҜunityд»Јз ҒиЈҒеүӘеј•з”Ёзҡ„зұ»еһӢзјәеӨұгҖӮеӨ„зҗҶж–№жі•еҗҢдёҠгҖӮ

# йҒҮеҲ° MissingMethodException: AOT generic method isn't instantiated in aot module xxx й”ҷиҜҜ

иҝҷжҳҜеӣ дёәAOTжіӣеһӢеҮҪж•°е®һдҫӢеҢ–зјәеӨұеј•иө·зҡ„пјҢ

дёҖе®ҡиҰҒ

иҜ·е…ҲиҜҰз»ҶзңӢдёҖдёӢAOTжіӣеһӢйҷҗеҲ¶еҸҠеҺҹзҗҶд»Ӣз»Қ ж–ҮжЎЈгҖӮ

и§ЈеҶіеҠһжі•дёәпјҡй”ҷиҜҜж—Ҙеҝ—е‘ҠиҜүдҪ зјәеӨұе“ӘдёӘAOTеҮҪж•°е®һдҫӢеҢ–пјҢдҪ е°ұеңЁдё»е·ҘзЁӢйҮҢеҠ дёҠеҜ№иҝҷдёӘеҮҪж•°зҡ„и°ғз”ЁпјҢдҪҝеҫ—il2cppеңЁжү“еҢ…ж—¶иғҪз”ҹжҲҗиҝҷдёӘжіӣеһӢеҮҪж•°зҡ„д»Јз ҒгҖӮ дё»е·ҘзЁӢйҮҢд»»ж„Ҹең°ж–№еҠ дёӘиҝҷдёӘжіӣеһӢAOTеҮҪж•°и°ғз”ЁйғҪеҸҜд»ҘпјҢзӣ®еүҚдёҖиҲ¬йӣҶдёӯеҠ еҲ° RefTypes.cs иҝҷдёӘж–Үд»¶йҮҢгҖӮ

еҰӮжһңжҳҜasyncзӣёе…ід»Јз ҒеҸ‘з”ҹиҝҷз§Қй”ҷиҜҜ(е…·дҪ“и§ҒдёӢйқўзҡ„зӨәдҫӢ5)пјҢеҲҷжҳҜеӣ дёәзј–иҜ‘еҷЁдёәasyncз”ҹжҲҗдәҶиӢҘе№Ізұ»еҸҠзҠ¶жҖҒжңәеҸҠдёҖдәӣд»Јз ҒпјҢиҝҷдәӣйҡҗи—Ҹз”ҹжҲҗзҡ„д»Јз ҒдёӯеҢ…еҗ«дәҶеҜ№AOTжіӣеһӢеҮҪж•°пјҲеҰӮзӨәдҫӢ5дёӯvoid System.Runtime.CompilerServices.AsyncVoidMethodBuilder::Start<T>(ref T)пјү зҡ„и°ғз”ЁгҖӮз”Ёеёёи§„зҡ„AOTжіӣеһӢзҡ„и§ЈеҶіеҠһжі•еӨ„зҗҶе®ғеҚіеҸҜгҖӮдҪҶжңүж—¶еҖҷпјҢдҪ ж·»еҠ дәҶvoid System.Runtime.CompilerService.AsyncVoidMethodBuilder::Start<StateMachine>(StateMachine&)зҡ„жіӣеһӢе®һдҫӢеҢ–пјҢзҗҶи®әдёҠиғҪжіӣеһӢе…ұдә«еҚҙд»Қз„¶жҠҘй”ҷпјҢеҲҷеҺҹеӣ жҳҜдҪ зҡ„dllдҪҝз”ЁдәҶreleaseзј–иҜ‘жЁЎејҸпјҢзј–иҜ‘asyncж—¶з”ҹжҲҗдәҶеҖјзұ»еһӢзҡ„зҠ¶жҖҒжңәпјҲд№ҹе°ұжҳҜзӨәдҫӢдёӯзҡ„UIMgr+ShowUId__2жҳҜеҖјзұ»еһӢпјүпјҢиҖҢеҖјзұ»еһӢжҳҜж— жі•жіӣеһӢе…ұдә«зҡ„гҖӮе…·дҪ“жғ…еҶөжңүдёӨз§Қпјҡ

  • еҰӮжһңдҪ зҡ„зғӯжӣҙж–°dllзӣҙжҺҘжӢ·иҙқиҮӘLibrary/ScriptAssembliesзӣ®еҪ•гҖӮи§ЈеҶіеҠһжі•дёәеңЁUnity EditorзӘ—еҸЈжңҖдёӢйқўзҡ„зҠ¶жҖҒж ҸпјҢжңүдёҖдёӘиҮӯиҷ«жҢүй’®еҸҜд»ҘеҲҮжҚў Debug/ReleaseпјҢеҲҮжҚўжҲҗDebugгҖӮ
  • еҰӮжһңдҪ зҡ„зғӯжӣҙж–°dllдҪҝз”ЁдәҶзұ»дјјhybridclr_trialйЎ№зӣ®зҡ„HybridCLREditorHelperйҮҢзј–иҜ‘д»Јз Ғзј–иҜ‘еҮәзҡ„dllгҖӮеҲҷеӣ дёәдҪ е°‘еҠ дәҶscriptCompilationSettings.options = ScriptCompilationOptions.DevelopmentBuild;пјҢеҜјиҮҙзј–иҜ‘еҮәзҡ„dllжҳҜreleaseжЁЎејҸгҖӮ

е№¶дё”еҠЎеҝ…

жё…йҷӨдҪ зҡ„Library/ScriptAssembliesзӣ®еҪ•еҸҠе·Із»Ҹжү“еҢ…еҘҪзҡ„dllгҖӮ

# йҒҮеҲ° ExecutionEngineException: GetManaged2NativeMethodPointer not support. xxxx еҮҪж•°еҗҚ

иҝҷжҳҜеӣ дёәиҝҷдёӘAOTеҮҪж•°дёҺinterpreterд№Ӣй—ҙзҡ„жЎҘжҺҘеҮҪж•°дёҚеӯҳеңЁгҖӮиҜ·еҸӮиҖғ жЎҘжҺҘеҮҪж•° ж–ҮжЎЈиҝӣиЎҢеӨ„зҗҶгҖӮ

# йҒҮеҲ° "This icall is not supported by il2cpp at System.AppDomain.Load"

жңүдёӨз§ҚеҺҹеӣ 

  1. еҰӮжһңйқһiosе№іеҸ°пјҢеҲҷеӣ дёәжңӘжӣҝжҚўlibil2cppзӣ®еҪ•
  2. еҰӮжһңiosе№іеҸ°пјҢеӣ дёәiosе№іеҸ°е№¶дёҚд»Һжәҗз Ғзј–иҜ‘libil2cppпјҢиҖҢжҳҜдҪҝз”ЁйҡҸEditorжү“еҢ…зҡ„libil2cpp.aпјҢдҪ йңҖиҰҒжӣҝжҚўlibil2cpp.aдёәHybridCLRзҡ„зј–иҜ‘зүҲжң¬гҖӮзј–иҜ‘ж–№ејҸиҜ·зңӢIOS Build (opens new window)

# йҒҮеҲ° Unity: TypeLoadException: Could not load type 'System.Object' from assembly 'netstandard'

зӣ®еүҚж”ҜжҢҒ.net standard 2.0пјҢдҪҶиҰҒжұӮдё»е·ҘзЁӢжү“еҢ…з”Ё.net standardпјҢиҖҢзғӯжӣҙж–°dllжү“еҢ…еҝ…йЎ»з”Ё.net 4.xгҖӮеҺҹеӣ жҳҜunityдҪҝз”Ё.net standardжү“еҢ…ж—¶пјҢдјҡиҮӘеҠЁеүҘйҷӨ.net standardзҡ„дҫқиө–пјҢзӣҙжҺҘдҫқиө–жңҖз»Ҳзҡ„dllпјҢеҜјиҮҙдё»е·ҘзЁӢзҡ„dllеҲ—иЎЁдёӯе®һйҷ…дёҠдёҚеӯҳеңЁnet standardиҝҷдёӘdllпјҢиҝӣиҖҢеҜјиҮҙзғӯжӣҙж–°dllеҠ иҪҪж—¶пјҢжүҫдёҚеҲ°жқҘиҮӘnetstandardзҡ„еҜ№иұЎгҖӮ

и§ЈеҶіеҠһжі•дёәзј–иҜ‘зғӯжӣҙж–°йғЁеҲҶж—¶api compatible levelеҲҮжҚўдёә.net 4.xжҲ– .net frameworkеҚіеҸҜгҖӮ

еҰӮжһңдҪҝз”ЁдәҶз¬¬дёүж–№dllеҪўејҸзҡ„жҸ’д»¶пјҢе№¶дё”жҸ’д»¶еҸӘж”ҜжҢҒ.net standardгҖӮе°ҶжҸ’д»¶ж”ҫеҲ°дё»е·ҘзЁӢд№ҹиғҪжӯЈеёёе·ҘдҪңпјҢдҪҶдёҚиғҪж”ҫеҲ°зғӯжӣҙж–°йғЁеҲҶгҖӮеҜ№дҫқиө–.net 4.xзҡ„жҸ’д»¶еҲҷж— йҷҗеҲ¶пјҢйғҪиғҪжӯЈеёёе·ҘдҪңгҖӮ

еҰӮжһңиҝҳеҸ‘з”ҹй”ҷиҜҜпјҢеҲҷеҲ йҷӨLibraryзӣ®еҪ•пјҢйҮҚж–°ж“ҚдҪңдёҖж¬ЎгҖӮ
